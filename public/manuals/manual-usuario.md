# Manual de Usuario

Este manual describe procesos funcionales del sistema para usuarios administrativos y operativos.

## 1. Inicio de sesión

1. Abre la aplicación en tu navegador.
2. Ingresa tu usuario y contraseña.
3. Si el sistema muestra un error de credenciales, verifica mayúsculas/minúsculas.

## 2. Navegación básica

- Usa el menú lateral para entrar a módulos (por ejemplo: Presupuesto, RRHH o Administración).
- Si no ves el módulo, puede ser por permisos del usuario.

### 2.1 Búsqueda rápida

Si la aplicación tiene un buscador, escribe 2–3 palabras clave del módulo y selecciona el resultado.

## 3. Ordenes de pago

Para registrar una orden de pago:

1. Entra al modulo de Administracion o Presupuesto segun tu perfil.
2. Abre la opcion **Ordenes de pago**.
3. Selecciona proveedor, tipo de documento y fecha de proceso.
4. Agrega los conceptos a pagar y valida montos.
5. Guarda la orden y confirma el estatus generado por el sistema.

Si no puedes completar el registro:

- Verifica que tengas permisos del modulo.
- Confirma que el proveedor este activo.
- Revisa que los montos no superen la disponibilidad.

## 4. Reportes

Para generar un reporte:

1. Entra al módulo correspondiente.
2. Selecciona el rango de fechas.
3. Haz clic en **Generar**.

Si el reporte no aparece:

- Revisa que el rango de fechas tenga registros.
- Verifica permisos del usuario.

## 5. Preguntas frecuentes

### 5.1 ¿Que hago si el asistente no encuentra informacion?

Si el asistente responde que no hay suficiente información en el contexto del manual, indica:

- el nombre del módulo,
- el paso exacto donde te detuviste,
- y el mensaje de error (si existe).

Si el asistente no encuentra informacion:

- Indica el nombre del modulo.
- Indica la accion exacta que intentas realizar.
- Adjunta el mensaje de error si existe.
